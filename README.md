# m-site-archive

本仓库用于归档与发布多个独立的 HTML 页面。这些页面可能来源于不同时期、不同主题的静态内容，统一存放于此以方便浏览、检索与引用。

## 项目简介

m-site-archive 是一个以归档为目的的静态页面集合。仓库内不针对任何特定域名或网站，仅作为 HTML 文件的存储与展示空间。页面内容可能包括个人笔记、工具页面、临时展示页、信息卡片等，不构成持续性服务或产品。

## 目录说明

- `docs/` — 存放主要的归档 HTML 页面，可直接通过 GitHub Pages 或其他静态托管服务访问。
- `assets/` — 存放页面所需的公共资源（如样式表、图片、字体等），按需组织。
- `archive/` — 存放历史版本或不再更新的页面，保留原始结构。

> 目录结构可能随归档内容增加而调整，建议以仓库当前状态为准。

## 页面归档说明

- 每个 HTML 页面均以独立文件形式存在，文件名反映其内容或用途（如 `note-2024.html`、`tool-calculator.html`）。
- 页面之间无强关联，可单独访问。
- 部分页面可能包含外部资源（如 CDN 库），但不保证长期可用性。
- 归档页面可能不定期更新或移除，恕不另行通知。

## 维护说明

- 本仓库由个人或团队维护，主要目标为保持归档内容的可访问性。
- 不保证所有页面兼容所有浏览器或设备。
- 欢迎通过 Issues 提交问题反馈（如链接失效、资源丢失等），但不承诺即时修复。
- 不接受主动提交的页面新增请求，除非与归档目标一致且经过评估。

## 使用方式

- 直接访问 `https://<你的用户名>.github.io/m-site-archive/`（若已启用 GitHub Pages）查看内容列表。
- 或克隆仓库到本地，通过任意 HTTP 服务器打开 `docs/` 目录浏览。
